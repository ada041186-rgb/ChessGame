# ChessGame
A desktop multiplayer chess game that allows two players to play against each other over a Local Area Network (LAN).
## How to Play(Planned)
1. Build and run the application on two computers connected to the same local network (Same Wi-Fi or software applications that create virtual LAN).
2. **Player 1:** Clicks "Create Game".
3. **Player 2:** Enters Player 1's local IP address and clicks "Connect".
